# WebSocket Chat Application

A full-stack, real-time chat application built using modern web technologies.  
This project demonstrates backend–frontend separation, real-time communication using WebSockets, and a scalable, production-style project structure suitable for academic submission and professional portfolios.

---

## Project Overview

This application enables real-time messaging between users through a responsive web interface backed by a Node.js server.  
It is designed to showcase full-stack development skills, real-time systems, and clean architectural practices.

Key objectives of this project:
- Implement real-time communication using WebSockets
- Maintain clear separation between backend and frontend
- Follow modular, scalable, and maintainable architecture
- Demonstrate industry-standard development practices

## Installation & Setup

### Prerequisites
- Node.js (v16 or later recommended)
- npm
- Git

---

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/<your-username>/WebSocket-Chat-Application.git
cd WebSocket-Chat-Application
```
### 2️⃣ Backend Setup
```bash
cd chatapp
npm install
npm run dev
```
### 3️⃣ Frontend Setup
Open a new terminal:
```bash
cd chatapp-frontend
npm install
npm start
```
### ▶️ Running the Application
``` text
Start the backend server

Start the frontend application

Open the frontend URL in your browser

Begin real-time chat interaction
```
